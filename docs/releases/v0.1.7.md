RelayBench v0.1.7

更新内容：
- 本地透明代理升级为入口调度器，支持自动选路、fallback、限速、脱敏日志、缓存和实时观测。
- 增强 Responses、Anthropic Messages 与 OpenAI Chat Completions 的协议探测和回退顺序。
- 引入 CPA 风格的本地短缓存、模型列表缓存、并发请求合并和路由健康持久化。
- 数据安全测试套件补齐 Prompt Injection、RAG 数据污染、恶意 URL / 命令诱导与 Jailbreak 等场景。
- 优化托盘、悬浮 token 窗、透明代理页面、日志展开、单实例唤醒和启动日志中文编码。
- 清理测试工程和临时编译产物，源码仓库只保留项目源码、文档和发布配置。
